# LLKC Platformas Testēšanas Projekts 🛡️

Šis projekts ir izveidots, lai praktiski demonstrētu manas QA iemaņas, veicot manuālo testēšanu LLKC (Latvijas Lauku konsultāciju un izglītības centrs) publiskajām platformām.

## 🎯 Projekta mērķis
Veikt funkcionālo un lietojamības (UX/UI) testēšanu, lai identificētu nepilnības un uzlabotu lietotāju pieredzi.

## 📋 Testēšanas apjoms (Scope)
* **Platforma:** [llkc.lv](https://www.llkc.lv)
* **Ierīces:** * Desktop (Windows 11, Chrome/Edge/Firefox)
  * Mobile (Android, Google Pixel)
* **Testēšanas veidi:** Manuālā testēšana, Responsivitātes pārbaude, Formu validācija.
